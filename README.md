# sql2struct
mysql/postgresql database to golang struct for gorm model

# install
```shell
go install github.com/starfishs/sql2struct@latest
```



# usage
```shell
sql2struct --dsn="mysql://root:123456@tcp(localhost:3306)/test?charset=utf8mb4"
```

#  support
- [x] mysql
- [x] postgreSQL
